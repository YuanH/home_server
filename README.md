# Home Server

## Prerequisite

- github
- one password cli
- docker desktop
- cloudflare-tunnel

## How to Run

`chmod +x 88-start.sh && ./88-start.sh`

or

`chmod +x 255-start.sh && ./255-start.sh`

~~`docker compose -f docker-compose-255.yml up -d`~~
